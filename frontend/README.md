# Frontend — Restaurante Manager (bonito, sin build)

- HTML + CSS + JS puro (sin frameworks).
- Estilo moderno (dark, glass, gradientes) y UX con toasts y paginación.
- Consume el backend en `http://localhost:3000`.

## Uso
1) Asegúrate de que el backend esté corriendo (`npm run dev` en la carpeta `backend`).
2) Abre `index.html` en tu navegador (doble clic o con Live Server).
3) Crea restaurantes, añade platos y navega por páginas (3 por página).
