# Restaurate App Completa (Backend JSON + Frontend bonito)

## Estructura
- `/backend` — API Express con persistencia en `data.json` (sin MongoDB).
- `/frontend` — Interfaz moderna (HTML/CSS/JS) que consume la API.

## Cómo correr
1) Backend
```bash
cd backend
npm install
npm run dev
```
API en: `http://localhost:3000`

2) Frontend
- Abre `frontend/index.html` en tu navegador.
